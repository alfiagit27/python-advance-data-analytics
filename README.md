# Python Basics — Practice Programs

This repository contains simple Python programs created to practice fundamental programming concepts.

## Contents

The programs in this repository demonstrate:

* Conditional statements (`if`, `elif`, `else`)
* Loops
* Functions
* Dictionaries and lists
* File handling
* User input handling

## Files

* `atm.py` — Basic ATM simulation program
* `filehandle.py` — Reading CSV file data
* `filehandle2.py` — CSV data processing using dictionaries
* `helloname.py` — Function with default arguments
* `printstat.py` — Store user input in dictionary
* `salary15.py` — Salary condition checking program
* `sumof10num.py` — Sum of numbers using loop and built-in function

## Requirements

Python 3.x

## Run a Program

Example:

```
python atm.py
```

## Purpose

This repository is for learning and practicing Python programming basics.
